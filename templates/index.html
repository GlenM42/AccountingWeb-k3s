    Brief description of the Project

    The story is simple as a penny. As soon as I began getting some cash on hand, it was obvious that I had to record it. For quite some time (around two years) there were no problems with using paper. Yet, in September 2023 I noticed I made a significant error that would lead to big complications in case of an audit.

    That led me to creating this accounting system with some, if not all required checks, UI and here we are!
